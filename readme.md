# prepend-file-cli

> Prepend data to a file, creating the file if it not yet exists.

## Install

```
$ npm install -g prepend-file-cli
```

## Usage

```
Usage
  $ prepend [file] [content]

Example
  $ prepend testfile.txt AppendThis
  $ prepend testfile.txt "Append more"
```
